# 🐾 PawCalculator

Консольный калькулятор на Rust с поддержкой обычных и «кошачьих» операций.

## ✨ Возможности

- **Базовые арифметические операции**: `+`, `-`, `*`, `/`
- **Кошачьи операции**: `ฅ+`, `ฅ-`, `ฅ*`, `ฅ/` — выводят результат с лапками! 🐱

## 🚀 Быстрый старт

### Сборка проекта

```bash
cargo build --release
```

### Запуск

```bash
cargo run
```

### Пример использования

```
Введите первое число: 5
Введите оператор: ฅ+
Введите второе число: 3
5 + 3 = 8 Лапок
```

## 📦 Зависимости

- `ask_input = "0.1.0"` — ввод данных от пользователя
